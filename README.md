## 项目介绍
该项目基于node、npm和vue-cli工具进行开发，具备强大的功能和便捷的使用方式。 
该项目的主要特点是支持多种类型的内容渲染，包括链接、图片、视频和代码块。

## 项目设置
```
<!-- 安装依赖 -->
npm install

<!-- 运行项目 -->
npm run serve

<!-- build项目 -->
npm run build
```
